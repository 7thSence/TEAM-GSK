How to Create a 7 Series(Digital complaint)

When to Create 7 Series

  • When the current service order cannot resolve the issue.
  • If the customer has a new issue that needs to be tracked separately.
  • When a new service order is necessary for proper tracking.

When Not to Create 7 Series

  • If the issue is already covered under an existing service order.
  • If creating a new 7 series causes redundancy in the system.

How to Create 7 Series

  • Log into the service order management system.
  • Navigate to the "Create New Service Order" section.
  • Select the 7 series option from the dropdown menu.
  • Fill in the required details.
  • Assign the service order to the appropriate team.
  • Confirm and save the service order.
  • Notify the customer with the reference number.
