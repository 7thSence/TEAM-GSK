How to handle existing 8 series Service order

Status: In Process

Sub-Scenario: The SO is still open

Action

  • Within TAT:

    Change to Release to WFM/send to vendor and adjust ETR date + 2 days.

  • Post TAT:

    Change to Release to WFM/send to vendor and adjust ETR date + 2 days.

Status: Assign WIP

Sub-Scenario: The SO is still open

Action

  • Within TAT:

    Create interaction followup - Ask to wait for the TAT

  • Post TAT:

    Create digital complaint - Give ASAP TAT

Status: DOA Confirmed/DOA Approved

Sub-Scenario: If Technical report (TR) status is available

Action

  • Within TAT:

    Interaction follow up - if it's within 2 days of TR report - Give email ID and TR number, ask the customer to email

  • Post TAT:

    Create complaint follow up, post 2 days of TR approval - Give email ID and TR number, ask the customer to email

Status: DOA Confirmed/DOA Approved

Sub-Scenario: If Technical report (TR) status is not available

Action

  • Within TAT:

    Complaint followup(7 series) - ASAP resolution to wait

  • Post TAT:

    Complaint followup(7 series) - ASAP resolution to wait

Status: Parts pending

Sub-Scenario: Always check the change history of previous SO, if the new value status is 'parts pending'

Action

  • Within TAT:

    Interaction follow up - Ask the customer to wait 7-10 working days

  • Post TAT:

    Complaint followup(7 series) - Ask the customer to wait 7-10 working days

Status: Customer deferment

Sub-Scenario: Check the customer appointment date in dates column - TAT will be appointment date

Action

  • Within TAT:

    Interaction follow up - Just wait for the appointment date

  • Post TAT:

    Complaint followup(7 series) - We give customer as soon as possible appointment date

Status: Send To Vendor

Sub-Scenario: New value in change history is Send To Vendor

Action

  • Within TAT:

    Just wait for the TAT - Interaction follow up

  • Post TAT:

    Complaint followup(7 series) - We give customer as soon as possible TAT

Status: Request for cancellation

Sub-Scenario: If SO is within 30 days

Action

  • Within TAT:

    Interaction follow up - Inform the customer to wait

  • Post TAT:

    Interaction follow up - Inform the customer to wait

Status: Completed

Sub-Scenario: If customer comes for a different issue which is totally different from previous SO

Action

  • Within TAT:

    Create new SO - Share TAT as per SO

  • Post TAT:

    Create new SO - Share TAT as per SO

Status: Technical support required

Sub-Scenario: If new value is Technical support required

Action

  • Within TAT:

    Just wait for the TAT - Interaction follow up

  • Post TAT:

    Complaint followup(7 series) - We give customer as soon as possible TAT
