How to Create a 7 Series(Digital complaint)

When to Create 7 Series

  • When the current service order goes out of TAT (as per hours)
  • When PMS request goes out of Appointment date
  • When SO is open and TR report gets crossed 2 days

When Not to Create 7 Series

  1. SO status is still within TAT from the time of SO raised in change history - as per hours change history.
  2. SO is within the Appointment deferment date.
  3. If SO status as new value in change history is

    "Request for cancellation"

  4. When a 7 series already exists in the transaction history

Steps to Generate a Digital Complaint on an Existing Service Order (SAP CRM - from create follow up)

  • Category: Click on "Category" and then select Resq CC
  • Reference Date: Enter the same as Request End Date
  • Reference Number: Input the 8 series SO number associated with the existing service order
  • Department: Select Resq from the department list
  • Category (Type of Issue): Choose Repair or Installation based on the nature of the complaint
  • Sub Category: Select As per issue
  • Date (ETR for Digital Complaint): Calculate and enter the ETR date by adding 2 days from today
  • Notes (Problem Description): Add the CX VOC (Customer Experience Voice of Customer) details to provide more context
